## Truth Rule

No unverified claims. No clutter. Proof over puffery.

Every outward-facing claim must be verifiable. If it can't be backed up, it doesn't ship.

## Canva Brand Kit Status

- The Canva connector now has `brandkit:read` scope (reconnect resolved the earlier
  `Missing scopes: [brandkit:read]` error).
- The Canva API can only **read** Brand Kits, not create them. The official Brand Kit
  must be created once in the Canva UI (Brand → Brand Kit) using the values above.
- Once the kit exists, it can be applied automatically to future generated designs
  via the connector.
